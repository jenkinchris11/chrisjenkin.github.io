Method 1 - Heuristic Evaluation, in this method i will be evaluating the game UI based on Nielsen's Heuristics the questions will be as followed: Screen Layout - 1. Can the user exit the game at any time ?

Burden on the player - 2. How long did it take for the user to get used to the interface.

Navigation - 3. Can the user navigate through the level with ease ? (Movement, Camera)

Game Story Immersion - 4. Is the games story tied into the gameplay ?

Status and Score - 5. does the game inform the player of where they are going ?

Method 2 - The second method being used is surveys with yes or no answers, i hope to get peoples opinion on the user interface of BlastBack and what could have been improved, with these surveys i will be asking the following questions:
1.Is the layout of the UI easy to see ?

2.Would you change the way the health bar looks ?

- 3.Is the character damage icon too big ?

4.is the pause menu too small ?
